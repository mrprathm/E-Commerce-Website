# 🛍️ E-Commerce Website — SHOPLANE

A fully responsive **E-Commerce frontend website** built with pure HTML, CSS, 
and JavaScript — featuring product listings (Clothing & Accessories), 
image slider, shopping cart, product detail pages, and order placement — 
all powered by external APIs.

---

## ✨ Features

- 🏠 Home Page — hero image slider with promotional banners
- 👗 Clothing Section — Men & Women clothing products with brand & price
- ⌚ Accessories Section — smartwatches, earphones & more
- 🔍 Search — search bar for Clothing and Accessories
- 🛒 Shopping Cart — add/remove items, cart count badge
- 📄 Product Detail Page — individual product info & images
- ✅ Order Placed Page — order confirmation screen
- 📱 Fully Responsive — mobile, tablet & desktop ready
- 🌐 API Integration — product data fetched via external APIs

## 🚀 How to Run

**Option 1 — Open directly:**
1. Clone the repo:
```
git clone https://github.com/mrprathm/E-Commerce-Website.git
```
2. Open `index.html` in any browser ✅

**Option 2 — Live Server (VS Code):**
1. Install **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
